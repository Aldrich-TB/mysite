欢迎使用 Intrusion Detection and Forensics system


Intrusion Detection and Forensics system（web入侵检测与取证系统）是一套面向web应用程序的安全检测产品。这套系统基于Python-Django框架开发，web前端使用Bootstrap3布局。系统具有易于使用，环境依赖性低，可以快速迁移到各类操作系统环境下的特点。

该系统从监督学习的角度开发了一套防御系统，可以通过实验得到有效的识别。


Intrusion Detection and Forensics system（web入侵检测与取证系统） is a set of security detection products for web applications. This system is developed based on the Python-Django framework, and the web front end uses the Bootstrap3 layout. The system has the characteristics of easy to use, low environmental dependence, and can be quickly migrated to various operating system environments.

This system has developed a set of defense systems from the perspective of supervised learning, which can be effectively identified by experiments.
